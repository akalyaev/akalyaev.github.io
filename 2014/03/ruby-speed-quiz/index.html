Ruby speed quiz

Fri, Mar 28, 2014

For each case choose the fastest option.

Platform, which hosted this quiz was closed. So it will be read only for some time.

1. Range cover? VS include?

1)

('a'..'z').include?('f')

2)

('a'..'z').cover?('f')

3) both run with similar speed

2. blk.call VS yield

1)

def foo
  yield if block_given?
end
foo { puts "Hi from foo" }

2) both run with similar speed

3)

def bar(&blk)
  blk.call
end
bar { puts "Hi from bar" }

3. Hash [] VS fetch

1)

h = {}
h[:a] || 1

2)

h = {}
h.fetch(:a, 1)

3) both run with similar speed

4. super with OR without arguments

1)

class Parent
  def bar(a, b)
    puts "#{a} - #{b}"
  end
end

class Child < Parent
  def bar(a, b)
    super(a, b)
  end
end
c = Child.new
c.bar(1, 2)

2) both run with similar speed

3)

class Parent
 def bar(a, b)
   puts "#{a} - #{b}"
 end
end

class Child < Parent
 def bar(a, b)
   super
 end
end
c = Child.new
c.bar(1, 2)

5. define_method VS class_eval (definition, NOT call speed)

1)

class A
  100.times do |i|
    define_method("foo_#{i}") { 10.times.map { "foo".length } }
  end
end

2)

class B
  100.times do |i|
    class_eval 'def bar_#{i}; 10.times.map { "foo".length };
end'
  end
end

3) both run with similar speed

Check yourself:

  1. 2
  2. 1
  3. 3
  4. 2
  5. 1
